Install CLI

Requirements

  • Windows10, Mac OSX, Linux
  • Node.js v.12.1.2 or higher

Install

Mac (Homebrew)

$ brew install Cliché-cli

NPM

$ npm install Cliché-cli

Yarn

$ yarn install Cliché-cli
